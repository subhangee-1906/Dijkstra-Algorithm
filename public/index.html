About the Visualizer

This tool provides an interactive way to understand Dijkstra's Algorithm, a fundamental concept in graph theory. Visualize how the algorithm finds the shortest path in a weighted graph, step by step. Customize the grid, set start and end points, add walls, and watch the magic happen!

Perfect for students, educators, and algorithm enthusiasts!
